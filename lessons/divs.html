Introducing <div>s

Divs are a HTML tag. Recall the different tags we learnt in the HTML node. We’re introducing them here in the CSS node as they are used heavily with CSS styling. You can think of DIV as a Division, or a container. It essentially allows us to section off our work in specific ways. Let’s dive into an example so this makes more sense.

I want to divide the page into 3 different colours. A yellow block, a black block, and a pink block.

To do this, in CSS we’re going to create 3 separate classes, and apply these to 3 different divs. Let’s take a look.

  1. Define the yellow block class in css.
    So in your style.css sheet: 
    .yellowBlock {
  background-color: yellow; 
}
    background-color - as is self explanatory in the name, defines a colour for the background of this class.
    yellowBlock is the name of the class. You set this. It is preceded by a full stop.

  2. Call upon this class in your HTML document.
    Code <html>
    <head>
    <title>Fun with divs!</title>
    <link rel="stylesheet" href="style.css" />
    </head>
    <body>
    <div class="yellowBlock">
    This block should be yellow
    </div>
    </body>
    </html>
    Fun with divs!
    This block should be yellow


Our div tag starts the division. It also sets the class to be yellowBlock, that we defined earlier. The /div tag ends the division, and therefore the yellow colour. Amazing that’s your first division complete!

Let's try again with a different division:
  1. Define a black block in CSS 
    .blackBlock {
  background-color: black; 
}

  2. Call upon this class in your HTML document.
    Code <html>
    <head>
    <title>Fun with divs!</title>
    <link rel="stylesheet" href="style.css" />
    </head>
    <body>
    <div class="yellowBlock">
    This block should be yellow
    </div>
    <div class="blackBlock">
    This block should be black?
    </div>
    </body>
    </html>
    Fun with divs!
    This block should be yellow
    This block should be black?


Do you see any issues here? Perhaps the question should be is there anything that you don’t see?
The text is not visible. This is because the text colour is set to black by default. We can change this in our CSS file too! You need to modify the ‘color’ property. Like so:

Code - style.css .yellowBlock {
background-color: yellow;
}
.blackBlock {
background-color: black;
color: white;
}
