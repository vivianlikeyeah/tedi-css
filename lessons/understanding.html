Understanding CSS

CSS describes how HTML elements should be displayed. The basic structure of CSS consists of a selector followed by a declaration. That declaration is made up of a property and it’s value.
Consider the following: 

h1 {
    color: yellow;
}

Let’s explain what’s happening here.


Element Explanation
h1 The selector, tells CSS which element(s) to target (in this case, <h1> tags).
color: yellow; The declaration, what the CSS is actually doing.
color The property of the declaration.
yellow The value of the declaration.
