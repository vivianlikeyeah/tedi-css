The following tools are very rarely ‘taught’ in a higher education setting (particularly within engineering degrees). However, they’re crucial in understanding how to develop visually appealing and aesthetic websites (as well as presentations, and other visual communication tools in general).

Finding the perfect shade

If you ask me, the two colours we selected in the previous example aren’t the most visually appealing. What if I want my shade of blue to be slightly lighter? Slightly darker? A little more green in there?

An easy way is to refer to the following website:
https://www.w3schools.com/cssref/css_colors.asp
As the website states, it highlights all of the colour names that are supported when coding in CSS.

The sky’s the limit

Actually, 16,777,216’s the limit. You can use many more colours than those in that list. In order to do so, you simply need to specify either an RGB value or a HEX value (of which there are 16,777,216 options).

How do you determine the HEX or RGB value of a colour? For our purposes, using a simple tool on the internet will suffice.

If you google search, ‘Colour Picker’ the following tool should appear:

By scrolling through the colours, and moving the placement of your marker to determine brightness, you should be able to exactly specify the colour you would like, and obtain the relevant RGB or HEX code. These codes can be used instead of the names we’ve been using so far.

fif For example: The HEX Code for LightSkyBlue is #87CEFA. You can double check this using the colour picker tool. Type #87CEFA in the hex field and the colour will change. The RGB values for SpringGreen are 0, 255, 127.

If we replace our previous words with these new HEX and RGB values, we will see no change in the output.
Code - style.css h1 {
color: #87CEFA;
}
h2 {
color: rgb(0, 255, 127);
}

Copy-Cat

What if you’re scrolling through the internet, and you happen to see a particular shade that you REALLY like? Or maybe, you work for a company with really strict branding-guidelines and you need to use their specific shade of yellow? There are tonnes of scenarios where know the exact HEX code of a colour you see is useful.

There are several software options available that can help you with this. For example: Mac systems include a program called Digital Colour Meter.

When you open digital colour meter, you can hover over any colour on your computer screen and it will tell you the exact RGB values.
